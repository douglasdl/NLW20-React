# NLW20-React

## Server

Create the Project:
```sh
npm init -y
```

Install the dependencies:
```sh
pnpm i typescript @types/node -D
```

Create the tsconfig:
```sh
npx tsc --init
```

- [tsconfig bases](https://github.com/tsconfig/bases?tab=readme-ov-file)

Install dependencies:
```sh
pnpm i fastify @fastify/cors fastify-type-provider-zod zod
```
